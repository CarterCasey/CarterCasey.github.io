TCP

Connection Management

  • Uses segments of max size 1460

  • Header:

    • Src and dest ports
    • Checksum
    • Seq No - position of first byte of segment
    • Ask No - seq no of next byte
    • Receiver Window - how many bytes it can handle

  • Hand Shake:

    • connect() sends SYN, holds initial sequence no (randomly generated)
    • server sends SYN-ACK, ack no is isn + 1, seq no is again random. Also contains receiver window.
    • client sends ACK with server's isn + 1 and receiver window (for client)

  • The hand shake gives us:

    • security, since there's a known connection for the stream.
    • reliability

  • Closing connection is done with FIN, FIN-ACK, ACK.

Reliability

  • Timeout

    • More than twice the propogation delay.
    • Sample RTT: most recent packet
    • Estimated Round Trip Time = .875 * previous ERTT + .125 * SRTT
    • DevRTT = (1 - β)DevRTT + β(SampleRTT - EstRTT)
    • Timeout = EstRTT + 4 * DevRTT
