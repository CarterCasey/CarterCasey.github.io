TCP Congestion Control

Loss

  • Timeout means there's probably congestion, because no packets are going through.
    • After timeout, set congestion window (cwnd) to 1.
  • Duplicate ACKs mean one packet got dropped, but others got through.
    • After 3 dup acks, old TCP (Tahoe) would set cwnd to 1.
    • New TCP (Reno) cuts cwnd in half after 3 dups.
    • New TCP also uses fast recovery - add 1 for every duplicate.
    • (Additive increase, multiplicative decrease).

Determining Throughput

  • During this phase, we hover between w and w/2 window sizes.
  • So 3w/4 / round-trip-time is the throughput.
