Networks - Lecture 15

Mar 24

Link Layer

  • Reliability - Forward Error Correction and Acknowledge/Request
  • MAC (medium access control)
    • Random Access (Aloha) - High collision rate
    • Timed Random Access (Slotted Aloha) - Can't collide with messages sent before
    • CSMA/CD - Carrier Sensing, Multiple Access / Collision Detection

Ethernet (Bus based)

  • When using Carrier Sensing, you must transmit for at least 2 * prop delay.
    • On ethernet, the minimum is 64 bytes. There's a physical limit on ethernet, namely for how far you can send the signal along the wire.
  • Upper bound of backoff intervals grow exponential with each collision.
  • Jamming?

Ethernet (Switched)

  • Only connect to a switch.

  • Link is full duplex (can send and receive at the same time).

  • Preamble: Series of seven bytes that indicate a synchronization, one byte to indicate the start of the message.

  • MAC addresses: Use them to bootstrap with DHCP. IP addresses aren't permanent.

  • ARP is used to populate a routing table with MAC addresses.

Quiz Questions

Quiz 1

  • Store-and-Forward switch stores packet before sending, Cut-Through just sends it through.
  • Delays:
    • Propogation: Distance / speed of light.
    • Transmission Delay: Connection throughput / message size.
  • For Store-and-forward, takes twice transmission delay.
  • Cut-through is only delayed by a byte (for the header of the packet).
