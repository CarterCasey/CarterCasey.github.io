Graph Coloring for Register Allocation

  • Instead of just spilling a variable, put it on the stack before all the other variables.
  • Passing by reference kills register allocation.

Instruction Selection

Register Restriction (ABI)

  • Caller Save - let callee mess with certain variables.
  • Callee Save - caller must get same values back in these registers.
  • Arguments: rdi, rsi, rdx, rcx, r8, r9
  • Stack and frame: rsp, rbp
  • Return: rax

x64 Peculiarities

  • Two-arg instructions: add x, y -> x = x + y
  • The cmp instruction: sets a bunch of those, then jump based on value.
  • Use cmove to do comparison eval (e.g. x = y < z)

Pre-coloring

  • If we know that certain values have to go in certain registers, we can just color those in the graph in the first place.
