HTML

SGML

The predecessor to/parent of HTML. First use of tags and elements.

  • Tag: "symbol" used to denote element in the text.
  • Attribute: additional piece of information attached to element.
  • Element: a piece of the text (like a paragraph, a block quote).
  • Contents: the text displayed within the element.

Tim Berners-Lee

Created HTML, basing it on SGML.

  • Started with very simple rules, so that it would be easy to use.
  • (Didn't actually think it would be as big a deal as it was.)

The language

  • Nesting format: start with tags, then (should) add and tags.
  • Can further nest with divs, spans, ps, etc.
  • contains meta-information about the document.
  • contains displayable elements and text, the actual content.
