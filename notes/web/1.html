Computer Communication

  • Every computer on the internet has an IP address.
  • Basic Network Connection Protocol (handshaking), A connects to B:
    1. SYN packet from A to B
    2. SYN-ACK packet from B to A
    3. ACK packet from A to B
    4. ... More communication
    5. FIN packet from A to B
