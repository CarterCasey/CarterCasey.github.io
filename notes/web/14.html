JQuery

  • Basically shorthand for frequently used javascript methods.
  • $ is the main indicator of jquery.
    • $("") accesses an element in the DOM.
    • $. allows for global(?) actions (e.g. $.GET).
  • Meant to stay compatible with all mainstream browsers (unlike javascript).
